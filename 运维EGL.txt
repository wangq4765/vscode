Command Return Code(1456最常用)
1：Ack is OK：确认没有问题
2：Busy, Try again (Command buffer is not empty)
3：CIM Mode is OFF
4：Command is already executing：命令已经在执行
5：Slot Value is not valid. (Mismatch or duplication)：层数值无效。(不匹配或重复)
6：Cannot perform now (Hardware problem)：现在无法执行(硬件问题)
7：Others. Detail information should be written in the Command Result's Comment.



Cassette State
1：No Cassette Exist
2：Waiting for Cassette Data
3：Waiting for Start Command
4：Waiting for Processing
5：In Processing
6：Process Paused
7：Process Completed

No Cassette: There is no cassette on the port.

Waiting for Start Command: Cassette Map has been received successfully, and waiting for Start Command from Master PLC or Operator.

Waiting for Cassette Data: A cassette is currently loaded in a port and waiting for Cassette Map Data from Master PLC or Operator. Equipment should report this event, after clamping the cassette and (if port has BCR) reading the cassette ID and (if port has mapping unit) sensing the existence of job for each slot.
等待卡夹数据:卡夹当前被装入端口，等待主PLC或操作员的卡带图数据。设备应在夹紧卡带并(如果端口有BCR)读取卡带后报告此事件磁带ID和(如果端口有映射单元)感知每个插槽的作业存在。
